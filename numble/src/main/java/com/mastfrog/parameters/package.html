Parameters API

This is a little library for managing parameters which come in the form of an untyped map object, and turning them into typed objects without hand-writing a lot of validation code.

It does two things:

  1. Generates a Java class from the parameters you list, via an annotation processor, so you have type-safe access to your parameters
  2. Provides a utility for validating the parameters before you instantiate one of those

Using this library comes in a few steps:

  • Write a wrapper for KeysValues that exposes the data you want to use at runtime. Essentially it's a Map<String,String> minus the mutators.
  • Annotate a type with @Params
  • Declare some parameters as its value, with @Param
    • Each param has a type
    • Each param has a list of validators from the built-in StringValidators enum from the Simple Validation project>
    • You can declare additional validators by type
      • These will be instantiated with Guice, so they can take any injectable object in a constructor annotated with @Inject
